# React + TypeScript の最小のテストを試すためのリポジトリ

## feature

- react
- typescript
- styled-components

### test tools

- react-addons-test-utils
- jest
  - ts-jest
  - @types/jest
- enzyme

## how to use

`$ npm i`

### start

`$ npm start`

### test start

`$ npm test -- --watch`
